# ⛏️ EndPoints

## 📂 Images

### GET All Images

```
GET api/ImageUpload
```

Retorna uma lista de todas as imagens

### GET Single Image

```
GET api/ImageUpload/:id
```

Retorna uma única imagem

### ADD Image

```
POST api/ImageUpload
```

Adiciona uma imagem

### DELETE Image

```
DELETE api/ImageUpload/:id
```

Deleta uma imagem
